# E-001-T21 Step A 构图占比分层标注（desktop/mobile）

## 目标口径
- 田地区：约 60%（±5%）
- 中景层：约 15%（±3%）
- 天空层：约 25%（±3%）

## 实际分层（按 CSS 分区）

### Desktop
- 天空层：`0% ~ 25%`（含太阳/云层）
- 中景层：`25% ~ 40%`（山丘/道路/围栏/庄园）
- 田地区：`40% ~ 100%`（九宫格+草地）

### Mobile
- 天空层：`0% ~ 26%`
- 中景层：`26% ~ 41%`
- 田地区：`41% ~ 100%`

## 结果
- Desktop：天空 25% / 中景 15% / 田地 60%
- Mobile：天空 26% / 中景 15% / 田地 59%

均在目标容差范围内。
